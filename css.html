Referência de CSS (Cascading Style Sheets)

Guia prático das principais propriedades e conceitos de CSS.

1. Seletores CSS

  • Seletor de elemento – Seleciona todas as tags. 
    p {
    color: blue;
}
  • Classe (.classe) – Reutilizável. 
    .botao {
    background: green;
}
  • ID (#id) – Único. 
    #menu {
    width: 200px;
}
  • Atributo 
    input[type="text"] {
    border: 1px solid #000;
}
  • Pseudo-classes 
    a:hover {
    color: red;
}
  • Pseudo-elementos 
    p::first-letter {
    font-size: 2em;
}

2. Box Model

  • width / height 
    div {
    width: 300px;
    height: 150px;
}
  • margin – Espaço externo. 
    div {
    margin: 20px;
}
  • padding – Espaço interno. 
    div {
    padding: 15px;
}
  • border 
    div {
    border: 2px solid black;
}
  • box-sizing 
    * {
    box-sizing: border-box;
}

3. Tipografia

  • font-family 
    body {
    font-family: Arial, sans-serif;
}
  • font-size 
    p {
    font-size: 16px;
}
  • font-weight 
    strong {
    font-weight: bold;
}
  • line-height 
    p {
    line-height: 1.6;
}
  • text-align 
    h1 {
    text-align: center;
}

4. Cores e Fundos

  • color 
    p {
    color: #333;
}
  • background-color 
    div {
    background-color: #f0f0f0;
}
  • background-image 
    body {
    background-image: url("bg.jpg");
}
  • opacity 
    img {
    opacity: 0.8;
}

5. Layout (Display, Position, Flexbox)

  • display 
    span {
    display: block;
}
  • position 
    .box {
    position: absolute;
    top: 10px;
    left: 10px;
}
  • Flexbox 
    .container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

6. Grid Layout

  • CSS Grid 
    .grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

7. Efeitos Visuais

  • border-radius 
    .card {
    border-radius: 10px;
}
  • box-shadow 
    .card {
    box-shadow: 0 4px 10px rgba(0,0,0,0.2);
}
  • transition 
    button {
    transition: background 0.3s;
}

8. Animações

  • @keyframes 
    @keyframes mover {
    from { left: 0; }
    to { left: 100px; }
}
  • animation 
    .box {
    animation: mover 2s infinite alternate;
}

9. Responsividade

  • Media Queries 
    @media (max-width: 768px) {
    body {
        background: #ddd;
    }
}
